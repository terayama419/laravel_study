PHP8、Laravel11（Laravel Blade + Livewire + Apple ）
帳票：ActiveReportsJS、VSJ
DB：Amazon Aourora PostgreSQL

プロンプトは以下のようなものを使っている。
【機能概要】
先ほどの食品のメーカー、仕入れ先を管理をしたいです。

【詳細仕様】
- メーカー、仕入れ先の新規登録
- メーカー、仕入れ先の更新
- メーカー、仕入れ先の削除
- メーカー、仕入れ先の一覧、検索機能、ページング(10件ずつ)
- 食品の「メーカー」「仕入れ先」はこのメーカー、仕入れ先で登録されたものから一覧を選んでDB登録したいです。
そのため、食品テーブルにメーカー名称や仕入れ先を入れるのではなく、それぞれのIDで管理したいです。
なお、メーカー、仕入れ先はそれぞれ別のテーブルで管理します。

【詳細仕様】
メーカーに関しては以下の項目が欲しいです
- 名称
- 住所
- 営業時間
- 連絡先電話番号
- 代表者名

仕入れ先に関しては以下の項目が欲しいです
- 名称
- 住所
- 営業時間
- 連絡先電話番号
- 代表者名
